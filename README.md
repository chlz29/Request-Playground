# Request-Playground

Playground for requesting an API

## Installation

Clone and run npm install to install the dependencies.

```bash
npm install // yarn install
```

## Start locally

To run the project locally, use the `start` command.

```bash
yarn start
```

## Unit Test

```bash
npm run test / yarn test
```
