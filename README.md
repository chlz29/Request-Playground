# ReactStarterKit

#### Repository Objective:

- playground for requesting an API

#### How to Start?

```
Cloning the repo
  git clone git@github.com:chlz29/Request-Playground.git

Development
  npm install
  npm run start / yarn start

Unit Test
  npm run test / yarn test
```
